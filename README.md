# Universe Auto Chess

Universe Auto Chess 是一款面向 PC 的自走棋 Roguelite 原型。玩家在一局 8 关的战役中选择阵营、部署单位、赢得战斗、获取奖励，并通过遗物、商店和难度修改器形成不同的局内构筑。

## 运行方式

本仓库包含已经构建好的 Windows 版本，不需要安装 Unity 即可试玩。

1. 下载或克隆本仓库。
2. 在 Windows 系统中打开仓库目录。
3. 双击 `AutoChess.exe` 启动游戏。
4. 不要删除 `AutoChess_Data/`、`MonoBleedingEdge/`、`UnityPlayer.dll` 或 `UnityCrashHandler64.exe`，它们是 Unity 运行时必需文件。

## 游戏概览

游戏围绕“战前部署 + 自动战斗 + 战后成长”的循环展开。玩家不直接操控单位攻击，而是在战斗前根据敌人阵容、棋盘位置和已有资源进行布阵。战斗开始后，单位会自动移动、索敌、释放技能并结算胜负。

核心体验包括：

- 自走棋式部署：在棋盘上放置近战、远程、辅助等不同定位的单位。
- Roguelite 战役推进：通过连续关卡构成一局完整体验。
- 战后奖励选择：胜利后获得单位、资源、遗物或其他成长机会。
- 地图节点决策：普通战、精英战、商店、遗物节点和 Boss 战穿插出现。
- 难度修改器：开局选择额外规则，改变资源、敌人强度或战斗压力。
- 伪 3D 表现：当前版本使用倾斜视角、地图背景、棋盘覆盖层和单位动态效果增强战斗观感。

## 当前版本内容

当前构建包含一条完整的 8 关样板战役：

1. 第 1 关：基础普通战，帮助玩家理解部署和自动战斗。
2. 第 2 关：普通战，开始加入远程压力。
3. 第 3 关：遗物节点，提供构筑方向选择。
4. 第 4 关：精英战，测试前排承伤与辅助应对。
5. 第 5 关：商店节点，用资源补强阵容。
6. 第 6 关：普通战，引入更明显的敌方机制。
7. 第 7 关：精英战，作为 Boss 前的强度预演。
8. 第 8 关：Boss 战，包含阶段压力和最终胜利结算。

完整流程为：

```text
主菜单 -> 选择阵营 -> 选择难度修改器 -> 地图节点
-> 战前部署 -> 自动战斗 -> 战后奖励
-> 商店 / 遗物 / 精英战 / Boss 战
-> 胜利或失败结算
```

## 操作说明

- 鼠标左键：选择单位、点击按钮、放置或确认操作。
- 鼠标拖拽或点击棋盘格：根据当前界面进行单位部署。
- 开始战斗按钮：部署完成后进入自动战斗。
- 返回/继续按钮：在地图、奖励、商店和结算界面推进流程。

开发测试版本内置调试面板：

- 按键盘左上角反引号键 ` 打开或关闭调试面板。
- 可用于跳关、增加能量、强制胜利、强制失败、打开地图和随机遗物。
- 该面板主要用于验收、调试和快速查看关卡内容。

## 游戏系统

### 单位与定位

单位按照战斗职责区分为前排、输出、远程和辅助等类型。不同单位拥有不同攻击距离、生命值、技能和战斗节奏。合理安排站位可以保护后排输出，也可以让前排更早吸收伤害。

### 棋盘与部署

战斗发生在固定棋盘上。玩家在战前阶段根据可用单位和敌人布局选择站位。当前版本强调清晰可读的棋盘覆盖层、选中反馈、部署反馈和战斗中的单位动态表现。

### 关卡与地图

战役地图由多个节点串联而成。普通战提供基础奖励，精英战提供更高压力和更高价值奖励，商店允许玩家消费资源调整阵容，遗物节点提供被动效果选择，最终 Boss 战检验整局构筑强度。

### 遗物与奖励

遗物是影响整局策略的重要被动效果。它们可能改变单位属性、资源节奏、战斗收益或阵容构筑方向。战后奖励用于补充单位、提升强度或扩大后续选择空间。

### 难度修改器

难度修改器会在开局改变本局规则，使同一套关卡产生不同挑战。它适合用于快速验证不同压力曲线，也为后续扩展更多 Roguelite 变体留出空间。

## 版本状态

这是一个可试玩的原型构建，重点用于展示核心玩法闭环、关卡流程、战斗部署、地图节点、奖励选择和伪 3D 视觉方向。它不是最终商业发行版本，后续仍可继续扩展正式美术、音频、更多单位、更多关卡和完整数值平衡。
